## 🧠 一、什么是音频采集

音频采集（Audio Capture）是指：
👉 把**现实世界的声音信号（模拟信号）**，通过麦克风转换成**数字信号（数据流）**的过程。
整个流程通常包括：

```
声音 → 麦克风 → 模拟信号 → ADC（模数转换） → 数字信号 → 采样缓冲区
```

---

## 🎛️ 二、核心参数

### 1️⃣ 采样率（Sample Rate）

* 定义：每秒采集声音的次数。
* 单位：Hz（赫兹），即每秒采样次数。
* 常见值：

  | 类型     | 采样率      | 说明            |
  | ------ | -------- | ------------- |
  | 电话语音   | 8 kHz    | 基础语音，仅限通话质量   |
  | 普通语音识别 | 16 kHz   | ASR常用，平衡质量与带宽 |
  | 音乐或TTS | 44.1 kHz | CD质量          |
  | 高保真录音  | 48 kHz   | 专业录音设备常用      |

> 🎯 ASR 模型（如 Fast-Whisper、SenseVoice）几乎都要求 **16 kHz 单声道 PCM 流**。

---

### 2️⃣ 位深度（Bit Depth / Sample Size）

* 定义：每个采样点用多少比特表示。
* 常见值：

  * 8-bit：音质差，用于低质量通信；
  * 16-bit：主流；
  * 24-bit / 32-bit float：高保真录音用。
* 越高位深度，动态范围越大（声音细节更多）。

---

### 3️⃣ 通道数（Channels）

* **1（单声道 Mono）**：语音识别最常用。
* **2（立体声 Stereo）**：音乐或高保真录制。

## 🔊 三、音频格式

| 格式       | 压缩    | 是否有损 | 说明               |
| -------- | ----- | ---- | ---------------- |
| WAV      | ❌ 无压缩 | 无损   | 最常用采集格式（ASR标准输入） |
| PCM      | ❌ 无压缩 | 无损   | 实际传给模型的原始数据      |
| MP3      | ✅ 有压缩 | 有损   | 体积小，不能直接用于ASR训练  |
| FLAC     | ✅ 压缩  | 无损   | 存储高保真音频          |
| OGG/Opus | ✅ 压缩  | 有损   | 网络传输常用（低延迟）      |

## 🧩 四、采集流程（典型音频输入管线）

以电脑麦克风为例：

```
麦克风
   ↓
声卡（内置ADC）
   ↓
音频驱动（如 ALSA / WASAPI / CoreAudio）
   ↓
操作系统音频API（如 PyAudio / Web Audio / PortAudio）
   ↓
应用程序读取音频流
```

在 Web、桌面、Python 端，采集方式略有不同：

| 平台                 | 常用技术                               | 示例                                    |
| ------------------ | ---------------------------------- | ------------------------------------- |
| Web 浏览器            | Web Audio API / MediaRecorder      | navigator.mediaDevices.getUserMedia() |
| Node.js / Electron | node-record-lpcm16 / mic           | 实时语音交互                                |
| Python             | PyAudio / sounddevice / torchaudio | 语音识别/采集脚本                             |
| C++ / 系统级          | PortAudio / ALSA / CoreAudio       | 专业应用                                  |

## ⚡ 五、采集时的关键技术点

* **缓冲区大小**：决定延迟。
  小缓冲区 = 低延迟（适合实时语音识别），但 CPU 压力大。
* **静音检测（VAD）**：判断讲话是否结束。
* **噪声抑制（Noise Suppression）**：降低环境噪声。
* **回声消除（AEC）**：防止录到自己的播放声音。
* **增益控制（AGC）**：自动调整音量。
